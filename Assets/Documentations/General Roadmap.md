* Blade Instance Data: This is a struct that will hold the theorical data that each blade should have. This doesn't take into account the mesh info or material info. This is the final product, it contains stuff like the height, the width, the curvature, and the wind reactions.

* The CurveFunction and Param: In this implementation, we are using the HermiteCurve unlike Ghost of Tsushima which uses a 2nd order Bezier Curve.
